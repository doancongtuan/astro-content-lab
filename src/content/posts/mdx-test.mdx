## What is MDX?

MDX lets you use components inside markdown. This is very powerful for content sites.

<Callout type="info" emoji="👾" title="Good to know">
  Astro supports MDX out of the box with the MDX integration.
</Callout>

## Why use MDX?

Normal markdown is limited. You can only use headings, lists, and code blocks.

<Callout type="tip" title="Pro tip">
  Use MDX when you need custom UI inside your content — like callouts, rating boxes, or comparison tables.
</Callout>

## Warning

<Callout type="warning" title="Watch out">
  MDX files use `.mdx` extension, not `.md`. Don't mix them up.
</Callout>
