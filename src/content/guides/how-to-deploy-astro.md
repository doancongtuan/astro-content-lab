## How to Deploy Astro to Vercel

Vercel is the easiest way to deploy an Astro site. It is free and takes less than 5 minutes.

## Step 1: Push your code to GitHub

Make sure your Astro project is on GitHub.

## Step 2: Connect Vercel to GitHub

Go to vercel.com, sign up with GitHub, and import your repo.

## Step 3: Deploy

Click deploy. Vercel will build and deploy your site automatically.

## Done

Every time you push to GitHub, Vercel will redeploy automatically.
