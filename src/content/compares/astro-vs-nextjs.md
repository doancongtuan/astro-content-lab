## Astro vs Next.js

Both are modern frameworks but built for different purposes.

## Astro

Astro is designed for content sites. It ships zero JavaScript by default.

Best for:
- Blogs
- Review sites
- Documentation
- Marketing pages

## Next.js

Next.js is designed for web applications. It is powered by React.

Best for:
- Web apps
- Dashboards
- E-commerce
- SaaS products

## Verdict

If you are building a content site, choose Astro. If you need a full web app, choose Next.js.
